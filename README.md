# JAD Advanced Math Quiz

تطبيق PWA تفاعلي لنماذج اختبارات رياضيات متقدمة، مبني فقط على الدروس والأمثلة الموجودة في ملفات:

- `1.PDF`
- `2.PDF`
- `3.PDF`

## المحتوى

يشمل النموذج 57 سؤالًا من نوع اختيار من متعدد وإجابة كتابية قصيرة. بعد كل سؤال يعرض التطبيق خطوات الحل والشرح مع مصدر السؤال من ملف الكتاب.

الموضوعات الحالية:

- الصيغ الأسية، الأس الصفري، التحليل إلى عوامل أولية، الجذور، العبارات العددية، المقادير الجبرية، والمعادلات.
- النسب المتكافئة، جداول النسب، المسائل الحياتية، والكسور/العشري/النسبة المئوية.
- زوايا الأشكال الرباعية، مساحة متوازي الأضلاع، المثلث، وشبه المنحرف. هذا هو القسم الأكبر في بنك الأسئلة الحالي.

## التشغيل من GitHub Pages

1. افتح المستودع: `https://github.com/AY2009MN/JAD`
2. من **Settings** اختر **Pages**
3. اختر الفرع `main` والمجلد `/ (root)`
4. بعد النشر سيكون التطبيق على:

```text
https://AY2009MN.github.io/JAD/
```
