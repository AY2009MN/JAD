# 🇬🇧 English Grammar Quiz - JAD

تطبيق PWA تفاعلي لاختبار قواعد اللغة الإنجليزية، موجه لـ **JAD**.

## 🚀 رفع التحديثات (المزامنة)

كلما أردت تعديل الملفات ورفع التغيير:

```bash
# 1. أضف كل التعديلات
git add -A

# 2. اكتب وصف التغيير
git commit -m "وصف التعديل"

# 3. ارفع إلى GitHub
git push
```

> **نصيحة:** يمكنك إنشاء ملف `sync.ps1` وتشغيله بدلاً من كتابة الأوامر يدوياً.

## 🌐 تفعيل GitHub Pages (نشر الموقع)

1. اذهب إلى المستودع: `https://github.com/AY2009MN/JAD`
2. اضغط على **Settings** ⚙️
3. من القائمة الجانبية اختر **Pages**
4. تحت **Branch** اختر `main` والمجلد `/ (root)`
5. اضغط **Save**
6. بعد دقيقة، الموقع سيكون متاحاً على:
   ```
   https://AY2009MN.github.io/JAD/
   ```

## 📲 استخدام التطبيق

- افتح الرابط في **Chrome** أو **Edge** على الهاتف أو الكمبيوتر
- سيظهر شريط تثبيت → اضغط **تثبيت** لاستخدامه كتطبيق بدون إنترنت

## 📚 القواعد المشمولة

1. Too / Not ... Enough
2. Present Passive (made of, used for, worn by)
3. Indefinite Pronouns
4. Modals (should / must / need to)
5. Present Perfect + Since / For
6. Present Continuous for Future
7. May, Might, Could, Can't, Must
8. First Conditional
9. Going to vs Will
